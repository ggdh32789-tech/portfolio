## 技术栈
- **纯静态网站**：HTML5 + CSS3 + 原生 JavaScript
- **无需服务器**：浏览器直接打开 index.html 即可
- **CDN 外部库**：
  - Google Fonts（Press Start 2P 像素字体 + Noto Sans SC 正文字体）
  - highlight.js（代码高亮）
  - Meting.js（网易云音乐播放器，歌单ID: 7250732567）
- **数据存储**：localStorage（留言板 + 语言偏好）

## 风格定义
- **配色**：深藏蓝 #1a1a2e 主背景 + 奶油白 #fdfaf3 卡片 + 藏红 #9b1b1b + 藏金 #d4a843 + 藏蓝 #1e3d59
- **风格**：藏式像素风 + 禅意留白
- **核心原则**：交互元素（链接/按钮/输入框）**绝不使用 clip-path**，全部用 box-shadow 做像素效果
